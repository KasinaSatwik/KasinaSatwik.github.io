Setting up

Language: C++, External libraries: OpenGL, IDE: CodeBlocks, . We made a few props used in the story using the OpenGL library. Namely, we used Bresenhams midpoint linedrawing algorithm, and midpoint circle drawing algoritm both of which were implemented using the OpenGL library in the code blocks IDE.

Bresenhams linedrawing

This algorithm uses integer arithmetic and is faster and more efficient that DDA. The main idea behind this alogrithm is to chose between two pixels, depending on whether the point lies inside or outside the curve. The variables x1,x2,y1,y2 indicate the endpoints of the line which is to be plotted, the variable m can take value of -1 or 1 depending upon whether the line's slope is positive or negative. Here, d is the decision parameter which helps us to take the decision of which pixel is to be colored.

Midpoint Circle drawing algorithm

This algorithm is used to draw circle given the center and the radius. It is also based on the usage of integer arithmetic and is the most optimal algorithm. The main idea behind the algorithm is we divide the circle into eight symmetrical parts and then we take the mirror image about the y=x;y=-x;y=0;x=0. The variable decision is the decision making parameter in our algorithm it is initialised to one minus the radius. We take the help of two variables DeltaE and DeltaSE which helps us to update the decision variable.
